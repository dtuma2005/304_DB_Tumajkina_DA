# Лабораторная работа 8 — CRUD-приложение для работы с БД клиники

## Описание
Веб-приложение для управления данными клиники с полным CRUD-функционалом: список врачей, график работы врачей, оказанные услуги.

## Структура проекта
Task08/
├── data/
│ └── task.db # База данных 
├── public/
│ ├── index.php # Список врачей
│ ├── doctor_form.php # Добавить / редактировать врача
│ ├── doctor_delete.php # Удалить врача
│ ├── schedule.php # График работы
│ ├── schedule_form.php # Добавить / редактировать запись графика
│ ├── schedule_delete.php # Удалить запись графика
│ ├── services.php # Оказанные услуги
│ ├── service_form.php # Добавить / редактировать услугу
│ └── service_delete.php # Удалить услугу
└── README.md

## Запуск
Запуск веб-сервера: php -S localhost:3000 -t public
Открыть в браузере: http://localhost:3000/
